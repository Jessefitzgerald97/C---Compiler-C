# C-- Compiler (C)

## Overview

This is a compiler the full C-- subset of the C programming language. This program takes as an input code written in the C-- language, checks the syntax and semantics, converts the code to assembly instructions, and produces an output.

## Features
  - Syntax and semantic checking
  - AST generation
  - Three-Address Instruction generation
  - Assembly code conversion

## How to Run

### Prerequisites
- GCC (or any C compiler)
- Linux / macOS terminal (or WSL on Windows)

make
